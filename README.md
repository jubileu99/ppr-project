> # ppr-project
> Trabalho para a disciplina de projeto de programas(PPR)


### Requisitos para executar o programa
* Maven
* OpenJdk 11
* Intellij

### Opcionais
* Scene Builder

### Executando o projeto

1. git clone https://github.com/jubileu99/ppr-project.git
2. cd ppr-project
3. mvn clean install
4. mvn javafx:run
